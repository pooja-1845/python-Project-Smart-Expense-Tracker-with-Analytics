# Smart Expense Tracker

A simple and smart Python project to **track your expenses**, categorize them, and analyze your spending habits.  

This project helps you **add expenses, view all records, analyze total spending per category, and find your top expense**. It uses Python features like **OOP, functions, exception handling, file handling, and advanced topics like lambda, map, reduce, and JSON**.

---

## Features

- Add expense with amount, category, and date.
- Save all expenses in a CSV file.
- Analyze spending per category and total spending.
- Find the top expense automatically.
- Save analytics in JSON format for future use.
